# Learning Playwright Javascript using Cermati Website
My first learning Playwright javascript using Cermati website.
Pre-requisite:
1. Install nodejs

How to run:
1. Open terminal in vscode then type:
````
npm init -y
npm install @playwright/test
npx playwright install
````
This is for setup the playwright.

2. Then for run steps, type:
````
npx playwright test --headed
````
